# **App Name**: VitalFlow

## Core Features:

- Hospital Login/Registration: Secure authentication and registration process for hospital users.
- Blood Bank Login/Registration: Secure authentication and registration process for blood bank admin users.
- Emergency Blood Request Submission: Hospitals can submit blood requests including patient name, blood group, units required and hospital name.
- Nearby Blood Bank Fetching: System automatically fetches nearby blood banks based on the hospital's location using Google Maps API.
- Real-time Blood Availability: Display real-time blood availability for each blood bank.
- Request Management: Blood banks can accept or reject blood requests from hospitals.
- Smart Blood Bank Prioritization: AI tool to prioritize nearest available blood bank based on urgency and availability, using location and real-time data.

## Style Guidelines:

- Primary color: Deep red (#B71C1C) to symbolize blood and urgency.
- Background color: Light gray (#F5F5F5), creating a clean and professional look.
- Accent color: Soft blue (#2196F3) for interactive elements, providing contrast.
- Body text: 'PT Sans', a sans-serif font that combines a modern look and warmth or personality, for readability.
- Headline text: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel.
- Use universally recognizable medical icons related to blood types, hospitals, and requests.
- Clean and structured dashboard layout for both hospitals and blood banks.
- Subtle animations for request status changes and updates.
